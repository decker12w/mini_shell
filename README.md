# JMShell

## Sobre 
Implementação de um interpretador de comandos básíco, baseado em sistemas Linux

## Etapas 

```bash
gcc -o shell ./shell.c
./shell
```

## Funcionalidaes 

```bash
$ prog
$ prog &
$ prog parâmetros
$ prog parâmetros &
$ prog [parâmetros] > arquivo
$ prog1 [parâmetros] | prog2 [parâmetros]
```
## Build In

```bash
$ jobs
$ exit
$ fg pid
$ bg pid
$ cd diretório
```
